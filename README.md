# 派对飞行棋 (Party Ludo) - 3-4人交互版

这是一个基于 React 开发的多人派对互动飞行棋 Web 应用。原项目为情侣双人版，现已重构为支持 3-4 名玩家参与的派对版本，特别适配了角色扮演机制（角色包括：丈夫、妻子、公牛、女伴等），为特定派对场景提供更有趣的互动与惩罚任务。

在线预览：[zainsu-bot.github.io/party-ludo/](https://zainsu-bot.github.io/party-ludo/)

## ✨ 功能特性

- **多人数动态支持**: 原生支持 3 人或 4 人同时参与游戏，满足派对场景。
- **派对角色扮演**: 玩家可被指定。
- **角色任务隔离**: 主题任务包可绑定特定角色。例如，“妻子”只会触发“通用”或“妻子专用”的任务卡，实现精准互动。
- **全新 S 型版图**: 升级为 8x5 的 40 格现代网格布局，支持多玩家同格避让显示。
- **随机指向互动**: 触发陷阱或追尾时，系统将随机在其他玩家中抽取一名“受害者”，增加多人的不可预见性。
- **沉浸式暗黑 UI**: 采用深色磨砂玻璃质感设计，灵感来源于现代 Vue 风格排版，视觉更具质感。
- **本地存储**: 游戏进度和自定义主题自动保存到浏览器本地存储（LocalStorage），随时继续游戏。

## 🛠️ 技术栈

- **前端框架**: [React](https://react.dev/) (v18)
- **构建工具**: [Vite](https://vitejs.dev/)
- **编程语言**: [TypeScript](https://www.typescriptlang.org/)
- **样式方案**: [Tailwind CSS](https://tailwindcss.com/)
- **图标库**: [Lucide React](https://lucide.dev/)

## 🚀 部署到 GitHub Pages

项目已内置 GitHub Actions 自动化部署流程。

1. 修改 `vite.config.ts` 中的 `base: './'`。
2. 将代码推送至 GitHub 仓库。
3. 在 Settings -> Pages 中将 Source 设置为 `GitHub Actions` 即可自动发布。

## 🎮 玩法说明

1. **配置阶段**: 在首页选择游戏人数（3-4人），并为每位玩家指定昵称、角色以及对应的任务主题包。
2. **任务管理**: 在“主题库”中可以针对不同角色新建专属主题，手动添加互动任务。
3. **游戏进行**: 轮流掷骰子移动棋子。由于是多人局，当某人触发任务时，系统会根据任务性质随机挑选互动对手。
4. **胜利条件**: 率先到达终点第 40 格的玩家获胜。
